# API Testing Portfolio

## Deskripsi
Pengujian API dilakukan menggunakan Postman pada public API Reqres (https://reqres.in).

## Tools
- Postman

## Jenis Pengujian
- Manual API Testing
- Automated API Testing (Postman Script)

## Endpoint yang Diuji
- GET /api/users
- GET /api/users/{id}
- POST /api/login
- PUT /api/users/{id}
- DELETE /api/users/{id}

## Artefak Pengujian
- Postman Collection (Manual)
- Postman Collection (Automation)
- API Test Case
- API Test Report

## Cara Menjalankan Automation
1. Import file automation-api-testing.postman_collection.json ke Postman
2. Klik Collection Runner
3. Jalankan seluruh request
4. Lihat hasil test pada tab Run Result
